# 平台支持说明

## 支持的平台

本项目仅支持以下平台：

### ✅ Android
- **最低版本**: Android 5.0 (API 21)
- **目标版本**: Android 14 (API 34)
- **架构支持**: armeabi-v7a, arm64-v8a, x86_64

### ✅ Windows
- **最低版本**: Windows 10 (1809 或更高)
- **架构支持**: x64

## 不支持的平台

以下平台已从项目中移除：

### ❌ iOS / iPadOS
- 原因：团队资源有限，优先支持 Android 和 Windows
- 如需支持：可以后续添加 iOS 平台支持

### ❌ macOS
- 原因：用户群体较小，暂不支持
- 替代方案：可使用 Web 版本（如果开发）

### ❌ Linux
- 原因：校园环境中 Linux 桌面用户较少
- 替代方案：可使用 Web 版本（如果开发）

### ❌ Web
- 原因：部分功能需要原生支持，Web 体验不佳
- 未来计划：可能会开发轻量级 Web 版本

## Android 开发环境

### 系统要求
- Windows 10/11, macOS, 或 Linux
- 至少 8GB RAM（推荐 16GB）
- 至少 10GB 可用磁盘空间

### 必需软件
1. **Flutter SDK** (3.0.0+)
2. **Android Studio** 或 **VS Code**
3. **Android SDK** (API 21-34)
4. **Java JDK** (11 或更高)

### 配置步骤

1. 安装 Flutter SDK
```bash
# 验证安装
flutter doctor
```

2. 安装 Android Studio
   - 下载地址: https://developer.android.google.cn/studio
   - 安装 Android SDK
   - 安装 Android SDK Command-line Tools

3. 配置环境变量
```bash
# Windows (PowerShell)
$env:ANDROID_HOME = "C:\Users\YourName\AppData\Local\Android\Sdk"
$env:PATH += ";$env:ANDROID_HOME\platform-tools"

# Linux/macOS
export ANDROID_HOME=$HOME/Android/Sdk
export PATH=$PATH:$ANDROID_HOME/platform-tools
```

4. 接受 Android 许可
```bash
flutter doctor --android-licenses
```

### 运行和调试

**使用模拟器**:
```bash
# 列出可用模拟器
flutter emulators

# 启动模拟器
flutter emulators --launch <emulator_id>

# 运行应用
flutter run
```

**使用真机**:
1. 开启开发者选项
2. 开启 USB 调试
3. 连接设备
4. 运行 `flutter run`

### 构建发布版本

```bash
# 构建 APK
flutter build apk --release

# 构建 App Bundle（推荐用于应用商店）
flutter build appbundle --release

# 输出位置
# APK: build/app/outputs/flutter-apk/app-release.apk
# AAB: build/app/outputs/bundle/release/app-release.aab
```

### 签名配置

创建密钥库：
```bash
keytool -genkey -v -keystore ~/key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias key
```

配置签名（`android/key.properties`）：
```properties
storePassword=<密码>
keyPassword=<密码>
keyAlias=key
storeFile=<密钥库路径>
```

更新 `android/app/build.gradle`：
```gradle
def keystoreProperties = new Properties()
def keystorePropertiesFile = rootProject.file('key.properties')
if (keystorePropertiesFile.exists()) {
    keystoreProperties.load(new FileInputStream(keystorePropertiesFile))
}

android {
    signingConfigs {
        release {
            keyAlias keystoreProperties['keyAlias']
            keyPassword keystoreProperties['keyPassword']
            storeFile keystoreProperties['storeFile'] ? file(keystoreProperties['storeFile']) : null
            storePassword keystoreProperties['storePassword']
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
        }
    }
}
```

## Windows 开发环境

### 系统要求
- Windows 10 (1809) 或更高版本
- 至少 8GB RAM（推荐 16GB）
- 至少 10GB 可用磁盘空间

### 必需软件
1. **Flutter SDK** (3.0.0+)
2. **Visual Studio 2022**
   - 必须包含"使用 C++ 的桌面开发"工作负载
   - MSVC v142 或更高版本
   - Windows 10 SDK

### 配置步骤

1. 安装 Visual Studio 2022
   - 下载地址: https://visualstudio.microsoft.com/zh-hans/downloads/
   - 选择"使用 C++ 的桌面开发"工作负载
   - 确保安装了 Windows 10 SDK

2. 验证环境
```bash
flutter doctor
```

确保显示：
```
[✓] Visual Studio - develop for Windows (Visual Studio Community 2022 17.x.x)
```

### 运行和调试

```bash
# 运行应用
flutter run -d windows

# 调试模式
flutter run -d windows --debug

# 发布模式
flutter run -d windows --release
```

### 构建发布版本

```bash
# 构建 Windows 应用
flutter build windows --release

# 输出位置
# build/windows/runner/Release/
```

构建后的文件结构：
```
build/windows/runner/Release/
├── opensna.exe          # 主程序
├── flutter_windows.dll  # Flutter 引擎
├── data/                # 资源文件
└── *.dll                # 其他依赖库
```

### 打包分发

**方式一：ZIP 压缩包**
```bash
# 压缩整个 Release 文件夹
Compress-Archive -Path build/windows/runner/Release/* -DestinationPath opensna_windows.zip
```

**方式二：安装程序（推荐）**

使用 Inno Setup 创建安装程序：

1. 安装 Inno Setup: https://jrsoftware.org/isinfo.php

2. 创建安装脚本 `installer.iss`：
```iss
[Setup]
AppName=山大网管会
AppVersion=0.1.0
DefaultDirName={pf}\OpenSNA
DefaultGroupName=山大网管会
OutputDir=installer
OutputBaseFilename=opensna_setup
Compression=lzma2
SolidCompression=yes

[Files]
Source: "build\windows\runner\Release\*"; DestDir: "{app}"; Flags: recursesubdirs

[Icons]
Name: "{group}\山大网管会"; Filename: "{app}\opensna.exe"
Name: "{commondesktop}\山大网管会"; Filename: "{app}\opensna.exe"

[Run]
Filename: "{app}\opensna.exe"; Description: "启动应用"; Flags: postinstall nowait skipifsilent
```

3. 编译安装程序：
```bash
iscc installer.iss
```

**方式三：Microsoft Store（可选）**

如需发布到 Microsoft Store，需要：
1. 注册 Microsoft 开发者账号
2. 使用 MSIX 打包
3. 提交审核

```dart
  // Android 特定代码
```

```dart

export 'platform_utils_stub.dart'
```

## 常见问题

### Android 问题

**Q: 构建失败，提示 Gradle 错误**
```bash
cd android
./gradlew clean
cd ..
flutter clean
flutter pub get
```

**Q: 真机无法连接**
- 检查 USB 调试是否开启
- 检查驱动是否安装
- 尝试更换 USB 线或端口

**Q: 应用闪退**
- 查看 logcat 日志
- 检查权限配置
- 检查混淆规则

### Windows 问题

**Q: 构建失败，提示找不到 Visual Studio**
- 确保安装了 Visual Studio 2022
- 确保安装了 C++ 桌面开发工作负载
- 运行 `flutter doctor` 检查

**Q: 应用无法启动**
- 检查是否缺少 DLL 文件
- 使用 Dependency Walker 检查依赖
- 确保 Visual C++ 运行库已安装

**Q: 窗口显示异常**
- 检查 DPI 设置
- 检查窗口大小限制
- 更新显卡驱动
